# 🔭 Дослідження алгоритмів сортування
*Проблема сортування* –  добре відома проблема програмування, з якою стикаються фахівці з обробки даних та інші розробники програмного забезпечення.<br> 
*Основна мета задачі сортування* – упорядкувати набір об'єктів у порядку зростання або спадання.<br> 
*Алгоритми сортування* – це послідовні інструкції, що виконуються для ефективного перевпорядкування елементів у списку або масиві в потрібний порядок.<br>
<br>
**Обрані алгоритми сортування :** MergeSort, InsertionSort, TimSort<br>
